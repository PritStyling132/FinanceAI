## Project Overview

This is a **Personalized Financial Advisor** application that uses:
- **Large Language Model (LLM)** - Ollama with llama3.2:3b for intelligent responses
- **RAG (Retrieval-Augmented Generation)** - For context-aware financial advice
- **Vector Database** - Qdrant for semantic search of financial knowledge
- **Real-time Market Data** - Alpha Vantage API for live stock data

## Summary

This AI-Powered Financial Advisor provides:

1. **Personalized Recommendations**: Based on user's risk profile, age, income, and goals
2. **Real-time Market Data**: Via Alpha Vantage API integration
3. **Intelligent AI Responses**: Using RAG pipeline with Ollama LLM
4. **Smart Fallback**: Specific stock/fund recommendations when LLM is unavailable
5. **Comprehensive Financial Tools**: Portfolio tracking, goal setting, budget management
6. **Secure Authentication**: JWT-based with bcrypt password hashing

The system is designed to work even when the LLM is not available, ensuring users always get useful financial guidance.
